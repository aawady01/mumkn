# ممكن (Mumkn)

> تطبيق أندرويد عربي متخصص في دعم التعافي من إدمان الإباحية والعادة السرية

---

## نظرة عامة

**ممكن** هو تطبيق تعافٍ سلوكي مبني على أسس علمية (CBT + ACT)، يساعد المستخدم العربي على:

- فهم محفزات السلوك القهري
- تتبع التقدم اليومي
- التعامل مع الرغبة فور حدوثها
- بناء عادات صحية بديلة
- الوقاية من الانتكاس

> التطبيق **ليس أداة تشخيص طبي**، بل نظام دعم ذاتي وتعافٍ سلوكي.
